About This System

Goal

This system was built to make it easier for students to access and monitor real-time bus availability. By digitizing the schedule, it eliminates the need to download multiple PDF versions, ensuring everyone always has access to the single, most up-to-date timetable without any confusion.

Official Announcement

According to the Majlis Perwakilan Pelajar (MPP), this bus schedule is valid from:
30 Mac 2026 hingga 21 Jun 2026.
